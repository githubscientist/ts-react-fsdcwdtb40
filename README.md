# React Application

npm init -y: To intialize the project to be a node project. This will create a package.json file.

to install all the dependencies or devDependencies listed in package.json file, we can run npm install or npm i

to install a specific package:

npm install package_name

or

npm i package_name

or to install a specific package as a dev dependency:

npm install package_name --save-dev

or

npm i package_name -D

package.json: This file contains metadata about the project, including its dependencies, scripts, and other configurations.

package-lock.json: This file is automatically generated when you run npm install. It contains a detailed description of the dependency tree, including the exact versions of each package and their dependencies.

readme.md: This file is used to provide documentation for the project.

.gitignore: This file will help us to exclude a certain files or folder from being tracked by git.
